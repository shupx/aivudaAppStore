# Backend

后端代码现已作为 `aivudaappstore` Python 包的一部分发布，实际入口位于：

- `aivudaappstore.backend.main:app`

接口前缀保持不变：

- 开发者管理接口：`/aivuda_app_store/dev/*`
- 公开商店接口：`/aivuda_app_store/store/*`
